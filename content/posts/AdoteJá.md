# ADOTE JÁ — CONECTANDO AMOR E ADOÇÃO 🐶🐱

## Sobre o projeto

O **Adote Hoje** é um sistema web desenvolvido para facilitar o processo de **adoção e doação de animais**, conectando pessoas que resgatam cães e gatos com quem deseja adotar.  
O projeto busca **reduzir o número de animais abandonados** e promover a **adoção responsável**, oferecendo uma plataforma intuitiva, acessível e segura.

---

## Principais Recursos

+ Cadastro e login de usuários com foto de perfil  
+ Recuperação de senha via email  
+ Cadastro e gerenciamento de animais disponíveis para adoção  
+ Upload de fotos dos animais  
+ Filtros de busca dinâmicos (espécie, raça, sexo, vacinação, castração)  
+ Interface responsiva e adaptável para celular, tablet e desktop  
+ Painel de administração simples e intuitivo  

---

## Como usar

1. Acesse o site [Adote Hoje](https://adotehoje.com.br) e crie sua conta.  
2. Faça login para acessar o painel de usuário.  
3. Se deseja **colocar um animal para adoção**, cadastre-o com fotos e informações completas.  
4. Se deseja **adotar**, use os filtros para encontrar o pet ideal.  
5. Entre em contato com o tutor pelo telefone disponível na página do animal.  

---

## Tecnologias utilizadas

+ **Backend:** Python 3, Flask, Flask-SQLAlchemy, Flask-Mail  
+ **Banco de Dados:** MySQL (PyMySQL)  
+ **Frontend:** HTML, CSS (Bootstrap 5.3.3), JavaScript, jQuery, Select2  
+ **Manipulação de Imagens:** Pillow  
+ **Servidor:** Nginx + Gunicorn (Ubuntu)  
+ **Hospedagem:** AWS EC2 + Certbot SSL
