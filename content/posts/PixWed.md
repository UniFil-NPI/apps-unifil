# PixWed

PixWed é uma plataforma web para criação de listas de casamento onde os presentes são convertidos em contribuições em dinheiro recebidas via PIX.

O objetivo central do projeto é resolver um “pain point” comum em casamentos: listas tradicionais amarram os noivos à loja e não permitem uso livre do valor recebido — enquanto o PixWed permite que todo o valor arrecadado seja destinado para onde o casal quiser, sem taxas, sem intermediários e sem amarras comerciais.

## Como funciona
- O casal cria uma conta.
- Configura sua lista de presentes (objetos, experiências, prioridades, etc).
- Cada item da lista tem uma sugestão de valor (ex.: “Contribuição para lua de mel”, “Ajuda no enxoval”, etc).
- Os convidados acessam a lista via um link público e podem contribuir por PIX.
- O valor vai direto para o casal (sem que o projeto segure o dinheiro).

## Principais princípios do projeto
- **Zero taxas** para o casal e para os convidados.
- **Privacidade**: dados não serão vendidos / compartilhados.
- **Liberdade financeira**: o casal usa o dinheiro como quiser.
- **Experiência simples** para o convidado (clique > abrir presente > pagar via PIX).

## Motivação
O projeto nasce de um contexto acadêmico de extensão, mas busca ser uma solução real para democratizar o processo de receber “presentes” de casamento sem depender de grandes varejistas.
