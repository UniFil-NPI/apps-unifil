# 🤝 APP SOLIDÁRIO – doações de alimentos com mais organização

## Sobre o projeto

O **App Solidário** é um projeto de extensão curricular desenvolvido para aproximar quem tem alimentos em boas condições para doar de quem precisa receber esse apoio. A proposta é tornar o processo de doação mais simples, organizado e acessível por meio de uma aplicação web.

O sistema funciona como uma ponte entre **doadores** e **beneficiários**, permitindo o cadastro de doações, a visualização dos alimentos disponíveis e o acompanhamento das solicitações feitas dentro da plataforma.

## Problema abordado

A fome e o desperdício de alimentos são problemas que muitas vezes acontecem ao mesmo tempo. Enquanto algumas pessoas e instituições precisam de ajuda, alimentos em boas condições podem acabar sendo descartados por falta de organização ou por dificuldade de encontrar quem possa recebê-los.

O App Solidário busca ajudar nesse processo, oferecendo um ambiente digital para registrar, organizar e acompanhar as doações.

## Principais recursos

- Cadastro e login de usuários.
- Perfis para doador, beneficiário e administrador.
- Cadastro de doações de alimentos.
- Registro de informações como tipo de alimento, quantidade, validade, descrição, endereço e urgência.
- Visualização de doações disponíveis.
- Solicitação de doações por beneficiários.
- Histórico das doações realizadas.
- Notificações sobre ações dentro do sistema.
- Área administrativa para acompanhamento de usuários e doações.

## Como usar

O usuário acessa a aplicação web, cria uma conta ou faz login. Depois disso, utiliza o sistema de acordo com o seu perfil.

O **doador** pode cadastrar alimentos disponíveis para doação. O **beneficiário** pode visualizar as doações cadastradas e solicitar os itens disponíveis. O **administrador** acompanha usuários e doações dentro da plataforma.

A navegação foi pensada para ser objetiva, facilitando o processo de cadastro, consulta e solicitação de doações.

## Tecnologias utilizadas

Backend: Python  
Frontend: HTML, CSS e JavaScript  
Banco de dados: SQLite  
Deploy: Railway

## Impacto esperado

O projeto busca contribuir para a redução do desperdício de alimentos e para o fortalecimento de ações solidárias. A plataforma também ajuda a organizar melhor o processo de doação, deixando registradas as informações de quem doa, do que foi doado e das solicitações realizadas.
