🚀 Interrogatio


🧩 Apresentação do Projeto

Interrogatio é uma plataforma web criada para auxiliar pessoas na preparação para entrevistas de emprego de maneira prática e interativa. O sistema simula entrevistas reais, permitindo que o usuário grave suas respostas em vídeo e receba uma análise sobre seu desempenho.

A proposta do projeto é oferecer uma ferramenta acessível para o desenvolvimento de habilidades interpessoais, comunicação e postura profissional, contribuindo para uma melhor preparação para o mercado de trabalho.


💼 Como Funciona

Durante a utilização da plataforma, o usuário grava um vídeo se apresentando e respondendo perguntas que simulam uma entrevista profissional. Após a gravação, o sistema realiza uma avaliação e apresenta relatórios contendo observações e feedbacks sobre a apresentação do candidato.

Entre os aspectos analisados estão:
🎙️ Forma de comunicação
📹 Postura diante da câmera
🧠 Organização das respostas
⏳ Administração do tempo
📈 Evolução do desempenho

O objetivo é proporcionar ao usuário uma experiência de treinamento próxima de uma situação real de recrutamento.


⚙️ Desenvolvimento

O projeto foi desenvolvido como um sistema web com foco em acessibilidade, simplicidade e praticidade de uso.

💻 Plataforma: Sistema Web
🧠 Linguagem: React.js, Django
🗄️ Banco de Dados: PostgreSQL
⚡️ LLM: Groq API
☁️ Hospedagem: Vercel


🌟 Principais Funcionalidades

✔️ Simulação de entrevistas de emprego
✔️ Gravação de respostas em vídeo
✔️ Feedbacks personalizados após cada simulação
✔️ Relatórios de desempenho
✔️ Interface intuitiva e de fácil utilização
✔️ Acesso remoto sem necessidade de instalação


👥 Público Beneficiado

🎓 Estudantes em fase de ingresso no mercado de trabalho
💼 Profissionais em busca de recolocação
🏫 Instituições educacionais e projetos acadêmicos
📚 Pessoas interessadas em aprimorar habilidades de comunicação


💡 Diferenciais do Projeto

O Interrogatio busca tornar o treinamento para entrevistas mais acessível e menos intimidador, permitindo que o usuário pratique quantas vezes desejar em um ambiente seguro e controlado.
Além disso, o sistema incentiva o autodesenvolvimento por meio de análises que ajudam o usuário a identificar seus pontos fortes e oportunidades de melhoria.


🏁 Considerações Finais

Mais do que uma plataforma de simulação, o Interrogatio é uma ferramenta de apoio ao crescimento profissional. O projeto demonstra como a tecnologia pode ser utilizada para desenvolver confiança, melhorar a comunicação e preparar pessoas para desafios reais do mercado de trabalho.
