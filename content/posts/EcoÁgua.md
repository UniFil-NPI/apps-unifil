# ECOÁGUA — CONSCIENTIZAÇÃO E CONTROLE DO USO DE ÁGUA 💧🌱

## 💡 Sobre o projeto

O **EcoÁgua** é um sistema desenvolvido para ajudar famílias e residências a **monitorar e reduzir o consumo de água**.  
Por meio de uma interface simples e interativa, o usuário pode cadastrar diferentes fontes de consumo (banheiro, cozinha, lavanderia etc.) e visualizar estimativas automáticas de gasto diário e mensal.

O objetivo principal é **aumentar a consciência ambiental**, reduzir desperdícios e fornecer **recomendações práticas** para melhorar o uso da água no dia a dia. 🌍

---

## 🔧 Principais Recursos

+ 💧 Cadastro de múltiplos pontos de consumo  
+ 📝 Inputs configuráveis:
  - Fonte de consumo (banheiro, cozinha, etc)
  - Vezes por semana (todos os dias, 3x semana, etc)
  - Tempo diário de uso
  - Quantidade consumida por dia
+ 📊 Cálculos automáticos:
  - **Consumo diário em litros**
  - **Consumo mensal em litros**
+ 🌱 Recomendações automáticas para redução do consumo  
+ 📱 Interface responsiva e intuitiva  
+ 📉 Comparação entre diferentes fontes de consumo  

---

## 📌 Como usar

1. Acesse a plataforma **EcoÁgua**.  
2. Adicione um ponto de consumo (ex.: “Banho”, “Lavar louça”).  
3. Preencha:
   - Frequência semanal  
   - Tempo diário de uso  
   - Quantidade utilizada  
4. O sistema calcula automaticamente:
   - 🔹 Consumo diário em litros  
   - 🔹 Consumo mensal em litros  
5. Adicione quantos pontos desejar para uma visão completa da residência.  
6. Confira as **recomendações** para economizar água e melhorar seus hábitos. 💦

## 🚀 Como rodar o projeto localmente

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/renan-munhoz/EcoAgua.git
cd EcoAgua
```

### 2️⃣ Backend (Python)

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 3️⃣ Frontend (Angular)

```bash
cd frontend
npm install
ng serve
```

### 4️⃣ Acesse a aplicação em:

```bash
http://localhost:4200
```
