# **NUTRIMATH**

## **Sobre o projeto**

O **NutriMath** é uma valiosa ferramenta no auxílio das refeições, uma vez que se trata de uma **calculadora de nutrientes simples e de fácil utilização**, porém **extremamente efetiva**.  

A calculadora tem o objetivo de **simplificar os cálculos nutritivos dos alimentos** e **agilizar a rotina de dietas**, oferecendo uma **ampla base de dados com alta precisão** em relação às calorias e macronutrientes dos alimentos.

---

## **Principais Recursos**

- Busca e seleção dos alimentos consumidos  
- Seleção das porções  
- Resumo dos macronutrientes e calorias diárias  

---

## **Como usar**

- Acesse o site e clique no botão **"Comece Agora"** para acessar a calculadora sem necessidade de login  
- Escolha os alimentos que consumiu da nossa base de dados  
- Indique o tamanho da porção em gramas  
- Acompanhe o resumo dos macronutrientes e calorias diárias  

---

## **Tecnologias utilizadas**

- **JavaScript**  
- **Node.js**  
- **Banco de Dados:** MySQL  
- **Hospedagem:** AWS
