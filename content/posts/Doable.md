# DOABLE - Planning Simplified

## Sobre o projeto

O Doable é um aplicativo mobile de lista de tarefas voltado para estudantes universitários. 
Permite criar, organizar e acompanhar tarefas com título, descrição, prazo e tags 
personalizadas, além de definir metas com progresso monitorado por período.

## Principais Recursos

- Definir tarefas com descrições e prazos.
- Organizar tarefas por tags criadas.
- Marcar tarefas como concluídas ou deletá-las.

## Como usar

- O usuário deve se cadastrar com email e senha ou realizar o login caso já tenha uma conta.
- O usuário pode criar uma tarefa informando título, descrição e prazo.
- Além disso, é possível editar, adicionar tags, deletar ou marcar a tarefa como concluído.

## Tecnologias utilizadas

- **Framework:** SpringBoot
- **Frontend:** Flutter
- **Backend/Serviços:** PostgreSql
